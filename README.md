# Password Fortress 密碼堡壘 - 密碼管理網站應用

網站連結：[Password Fortress](https://password-fortress.onrender.com)

備註：進入網站可能需等待 Render 完成部署，網站才會成功載入

## 功能介紹
- **密碼強度檢測**：即時分析並顯示密碼強度
- **密碼加密儲存**：採用 bcrypt 雜湊演算法保護使用者密碼
- **密碼自動生成**：根據使用者設定（長度、生成數量）隨機生成強度高的安全密碼
- **CRUD 操作**：支援新增、檢視、編輯與刪除密碼條目
- **密碼存儲管理**：依網站名稱、使用者名稱來儲存密碼
- **使用者認證**：提供註冊、登入與忘記密碼功能、並須經過兩步驟驗證才可對儲存的密碼進行更動

- **部署**：Render.com
